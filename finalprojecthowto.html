HOW TO: Make Images Buttons

Firstly, if you want to just create a simple image button that makes something happen, you must first create a <form> tag. Within the form, you can now start creating your button. Start off by typing in <input type="image". This tells the computer that you're going to be making an image button. You can also assign a class or id to the button to style it individually. After this, you would type in src="", and a link to your image would be in the quotations. Below is an example of a simple image button and the code for it.

Now, you have a button that doesn't do anything. In order to make it do stuff, we'll need to start typing in javascript. For this, I'll use my own code as an example. In my code, what I did with buttons was make it so they unhide divs on click.
The code for this using our above example would go like this:
var EmmaStone = document.getElementById("EmmaStone");
A quick reminder: that and the next line of example code are in javascript. What that line of code essentially does is establish "EmmaStone" as a variable. Next, you would want to start off on a new line with:
EmmaStone.addEventListener("click", function(){

}) Now, you just have to type in what you want the button to do on click. For me, I want the button to unhide a hidden div. If a div is hidden in the first place, that means it has its class set to something like class="hidden", and that class is set to display:none. Therefore, we would just want to simply change the class of the div on click, which would unhide it because its class wouldn't be set to "hidden" anymore. This means that I would type in: ExampleDiv.className=""; right in the empty line between the brackets.
