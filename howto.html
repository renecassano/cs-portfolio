How To Page

Creating a website requires a decent amount of knowledge about coding. Some of the things you need to know are: how to import images, how to create different headers and fonts, etc. In this page, I will be explaining how I used these various computer science tools.
One of the first aspects of creating a website you will probably encounter is headers. Headers are used to introduce different sections of a page, similar to how they are used in other mediums, e.g. newspapers Headers come in different sizes, the largest one being <h1> </h1> and the smallest one is <h6> </h6>. I used headers and titles for all the pages I made to better organize the information. Another vital aspect I used was <p> </p. It's used to create paragraphs and is the tool I used to construct the paragraphs you're reading right now. Just about all of these tags and tools are contained within <body> </body>, which is where all the visible words, pictures etc. on the website are stored in the code. Another integral part of creating webpages is images. Images are fairly simple to use in html. you start off with a tag like <img src="EXAMPLELINK.COM">. the EXAMPLELINK.COM space would be filled by whatever the url is for the image you want to use. The size of the image can also be changed within the tag. Typing in "width=x" and "height=x" accomplishes just this. Another aspect of the website I had to construct was the navigation bar. The navbar is established with the line <ul id="navbar">. inserting a line such as <li> <a href="example.html"> Example Page Name </a></li> underneath the navbar establishing line would then link what ever page I am currently making the navbar on to the page example.html, which would be called "Example Page Name" on the navbar seen by the viewer. after all li items are typed in, you end the tag with at the end of everything. The navbar also has to be copy and pasted on every tab you want to be included in your final website. Finally, you also need to create a css file to link to and customize the navbar. Once you know how to utilize all these tools, creating a website should be easy from here on out.
